# Django

Placeholder for Django learning projects.

## Getting started (when ready)

```bash
pip install django djangorestframework
django-admin startproject myproject .
python manage.py startapp api
python manage.py runserver
```

Create one subfolder per project, e.g. `blog-api/`, `todo-app/`, each with its own `manage.py` or use a monorepo layout you prefer.

Suggested first project: **REST API for a simple todo list** with Django admin and one `Todo` model.
